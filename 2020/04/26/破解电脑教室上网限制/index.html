破解电脑教室上网限制

1.MAC限制破解：
第一：让老师无法监控到你
首先，打开开始菜单–控制面板，选择windows防火墙，之后打开，点击不允许例外。之后确定，回到桌面
这样就可以让老师无法控制你啦，因为老师对你这个发送的数据已经被防火墙拦截，但是你这个客户端发送给老师的数据老师还是收的到滴。

第二：寻找目标，准备破解
打开开始-运行-输入“ipconfig” （去掉引号），会有你内网IP的地址，找到 Default Gateway项，会对应你的交换机的IP地址，这个就是我们的目标，如果交换机不行上网，那就找其他的，因为只有你和老师的在进行链接，除了老师的MAC没有其他人了。
大多数的上网限制都是MAC破解，一般交换机是允许上网的，如果你的电脑教室是对于端口的保护或者是狠心到把交换机也不让上网的话你就蛋疼了。
第三：开始突破
继续在cmd里面输入命令 ping+空格+ Default Gateway的IP地址，之后输入arp -a 就会有一个连接是指向192.168.1.1 的 后面跟着一个 ec-88-8f-39-48-80类似这样的格式，就是他，把它记下来。
之后右键属性网上邻居，之后本地连接也是右键属性，找到配置(C)点进入 选择高级，找到Network Address（网络）地址，在右边的值第一个圆圈打上勾，之后把刚刚arp -a的那个地址复制进入，确定，OK，可以上网了。
二：DNS限制
这个也是最简单的，直接在网上邻居-右键-属性，之后本地连接，属性，TCP/IP协议，点进去。修改一下DNS就行了，这个比较好破解，就不多说了。
